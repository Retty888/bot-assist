## Мобильный вид

```
┌──────────────────────────────┐
│ Header                       │
├──────────────────────────────┤
│ Загрузка сигнала             │
├──────────────────────────────┤
│ Статусы заказов (фильтр)     │
├──────────────────────────────┤
│ Трекеры                      │
├──────────────────────────────┤
│ Сводка + быстрые действия    │
└──────────────────────────────┘
```

Основной акцент сделан на сводке статусов и быстрых действиях слева, рабочей области форм и заказов справа, а также явных индикаторах состояний для трекеров и заказов.
